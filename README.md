# IT Support and Help Desk Lab Portfolio

**Author:** Nnamso Mkpong
**Last Updated:** March 2026

---

This is my personal IT support lab portfolio. Each lab was built and documented by me as part of my preparation for help desk, service desk, and desktop support roles.

The repository contains 20 labs covering the skills that appear most consistently in IT support job descriptions. Every lab folder holds three things: the full written walkthrough in README.md, a ticket note written in business style, and a screenshots folder with annotated evidence from the actual lab session.
