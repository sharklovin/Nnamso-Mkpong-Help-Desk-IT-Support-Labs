# IT Support and Help Desk Scenarios
**Author:** Nnamso Mkpong
**Last Updated:** March 2026

---

This is my personal IT support lab portfolio. Each lab was built and documented by me. Everything here i can do them with my eyes closed.

The repository contains 20 labs covering the skills that appear most consistently in IT support and Help Desk job descriptions. Every lab folder holds three things: the full written walkthrough in README.md, a ticket note written in business style and a screenshots folder with annotated evidence from the actual lab session.
